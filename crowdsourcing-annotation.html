Reducing external information in crowdsourcing labeling for salient object labeling

crowdsourcing, computer vision, object labeling, interface, AMT


Machine learning is to give computer systems the ability to "learn" with data. This time, we want to teach the computer to understand the images as we do. Like parents using alphabet cards teach babies, we train computers with datasets which consist of images with annotations. So, high-quality datasets are important to be used in training the algorithm.

What we do in this project is test a new interface and demonstrate that it can improve the effectiveness and efficiency of the labeling work.

We collect images with annotations by crowdsourcing work. Will you feel annoyed if I ask you to annotate all the objects in the yellow bounding box in the picture below?



The fact is crowdsourcing workers always be asked to label a very complex image with multiple objects. So, we change the display of the image by adding the light grey overlay to reduce redundant information and highlight the object needed to be labeled.



To test the new interface of our work, we post HITs on the crowdsourcing platform -- Amazon Mechanical Turk(AMT) to experiment our method. We do the experiment using 3 groups. The control group uses images with the bounding box. The two experiment groups use images with overlays. The opacities of the overlay are set to be 80% and 100%. The results of the experiments demonstrate that add the overlay with 80% opacity can improve the efficiency and effectiveness of labeling objects in images.
